============================================================
  STEP 2: Common Crawl harvest (~200M-1B words, ~80 min/crawl)
============================================================
  # Single crawl (latest) — ~38K docs, ~80 min
  python scripts/data_scraper/cc_bd_harvest.py \
      --out saved/data/raw/cc_bangla/ \
      --per-domain-limit 15000 \
      --workers 16

  # Add more crawls later (append mode)
  python scripts/data_scraper/cc_bd_harvest.py \
      --crawl CC-MAIN-2026-17 CC-MAIN-2026-12 CC-MAIN-2026-08 \
      --out saved/data/raw/cc_bangla/ \
      --per-domain-limit 15000 \
      --workers 16 \
      --append

============================================================
  STEP 3: Dedup & check quality

  python scripts/dedup_corpus.py saved/data/raw/cc_bangla/
